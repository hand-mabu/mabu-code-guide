# MabuDoc

MabuDoc 是基于 [Node](https://nodejs.org/) & [Hexo](https://hexo.io/zh-cn/docs/) 搭建的静态页面框架

> 可以帮助企业快速搭建文档系统。

* [Demo演示&使用教程](https://hand-mabu.github.io/hexo-mabu-doc/)
